Ejercicios JavaScript resueltos

Archivos incluidos:
- variables.js
- strings.js
- arrays.js
- if-else.js
- for.js
- funciones.js

Cómo ejecutar desde PowerShell:

1) Descomprimir el ZIP.
2) Entrar a la carpeta:
   cd ejercicios_js_resueltos

3) Ejecutar el archivo que quieras probar:
   node variables.js
   node strings.js
   node arrays.js
   node if-else.js
   node for.js
   node funciones.js

Nota:
Algunos ejercicios piden usar alert(). Como alert() existe en el navegador pero no en Node.js,
los archivos incluyen una función mostrarMensaje() que usa alert() si está disponible
y console.log() si se ejecuta desde Node.js.
