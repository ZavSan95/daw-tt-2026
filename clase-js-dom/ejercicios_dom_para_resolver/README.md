# Ejercicios DOM - JavaScript

Cada carpeta contiene un ejercicio independiente.

Estructura de cada ejercicio:

- `index.html`: archivo HTML base.
- `style.css`: estilos mínimos para visualizar el ejercicio.
- `script.js`: archivo donde deben resolver el ejercicio. Incluye comentarios con el enunciado.

## Cómo trabajar

1. Abrir la carpeta del ejercicio.
2. Abrir `index.html` en el navegador.
3. Editar solamente `script.js`, salvo que el docente indique lo contrario.
4. Probar el resultado en el navegador.
